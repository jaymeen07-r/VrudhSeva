# Elder User (Core User)

## Overview
The **Elder** is the primary and core user of the *VrudhSeva* system.  
All services, permissions, and interactions revolve around the Elderâ€™s needs, safety, and well-being.

## Role & Importance
- Central user of the platform
- Receives care, assistance, and monitoring
- Other users (Family, Caretaker) are linked to an Elder account

## Key Responsibilities
- Use basic app features (communication, alerts, requests)
- Receive health, safety, and service support
- View assigned caretakers and family members

## Key Features
- Emergency alerts (SOS)
- Health and daily activity tracking
- Communication with family and caretaker
- Simple, senior-friendly UI

## Access Level
- Limited and protected access
- Cannot modify system or administrative settings
- Actions are monitored for safety

## Dependencies
- Linked with:
  - Family Members (Primary & Secondary)
  - Caretaker
  - Admin for system-level management

## Notes
- UI and UX are designed for ease of use
- Accessibility and safety are the top priorities
