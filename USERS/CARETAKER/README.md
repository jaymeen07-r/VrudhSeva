# Caretaker User

## Overview
The **Caretaker** is responsible for providing daily assistance and support to the Elder.  
They act as the on-ground support user.

## Role & Responsibilities
- Assist Elder with daily activities
- Update health and care records
- Respond to alerts and emergencies
- Communicate with Family members

## Key Features
- Task and schedule management
- Health updates submission
- Emergency response handling
- Direct communication with Family
