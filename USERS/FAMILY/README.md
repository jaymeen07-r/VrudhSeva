# Family Member User

## Overview
Family members are trusted users connected to an Elder.  
They help monitor, communicate, and support the Elder remotely.

## Family Types
- **Primary Family Member**
  - Main decision-maker
  - Higher permissions
- **Secondary Family Member**
  - Support role
  - Limited permissions

## Key Responsibilities
- Monitor Elderâ€™s health and activity
- Communicate with Elder and Caretaker
- Receive alerts and notifications
- Provide approvals when required

## Key Features
- Real-time status updates of Elder
- Emergency notifications
- Communication tools (chat/call)
- Care reports and history access

## Access Level
- Higher than Elder
- Primary member has more control than Secondary
- No system or developer-level access

## Dependencies
- Must be linked to an Elder
- Interacts with Caretaker and Admin indirectly

## Notes
- Designed for trust and transparency
- Permission-based access control
