# User Onboarding Flow

This document explains how users are onboarded into the system.

## Step-by-Step Flow

### 1. Admin Creates Elder
- Admin registers Elder profile
- Basic details entered
- Unique Elder ID generated

### 2. Family Linking
- Primary Family member invited
- Secondary Family member added (optional)
- Permissions assigned automatically

### 3. Caretaker Assignment
- Admin assigns caretaker to Elder
- Care schedule configured

### 4. First Login
- Elder logs in with simple authentication
- Family and Caretaker receive access confirmation

## Validation Rules
- Elder must exist before linking others
- Only Admin can assign Caretaker
